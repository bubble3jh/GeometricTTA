# Notes directory

This directory is the shared “paper trail” for research decisions and reproducibility.

## Recommended files
- `plan.md` (use `notes/templates/plan.md`)
- `experiment_log.md` (use `notes/templates/experiment_log.md`)
- `retro.md` (use `notes/templates/retro.md`)
- `report_outline.md` (use `notes/templates/report_outline.md`)

## Rule of thumb
If a decision would be hard to reconstruct 2 weeks later, write it down here.
