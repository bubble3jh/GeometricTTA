# Implementation Specification: Projected Evidence-Gated BATCLIP

## Objective
Upgrade the existing BATCLIP TTA pipeline based on recent diagnostic findings. The core issues with the baseline are:
1. Optimizing global feature separation (InterMeanLoss) restores noisy, off-text dimensions, causing performance degradation.
2. Overconfident-wrong samples destroy prototype directions implicitly; they do not have large gradient norms but act as directional poisons.
3. In short-horizon regimes (N=1K, 5 steps), adaptation fails due to a lack of dynamics, but naive recirculation causes late-stage drift.

You need to implement three core methodological levers and a validation script.

---

## Component 1: Text-Projected Prototype Adaptation (Lever 1)
**Goal:** Restrict the InterMeanLoss to operate strictly within the subspace spanned by the text prototypes.

**Implementation Logic:**
1. Given text embeddings $Z \in \mathbb{R}^{d \times C}$ (where $d=512$, $C=10$), compute the orthogonal projection matrix $P_Z$ onto the column space of $Z$.
   - *Hint:* Use SVD or QR decomposition on $Z$ to get an orthonormal basis $U \in \mathbb{R}^{d \times k}$ (where $k \le C$).
   - $P_Z = U U^\top$.
2. In the `InterMeanLoss` computation module, before computing the separation between class prototypes, project the normalized image features $v$:
   - $v_\parallel = v P_Z$
3. Compute the prototype means and the separation loss *only* using $v_\parallel$. 
   - *Constraint:* Do NOT apply $P_Z$ to the features used for $L_{I2T}$ (matching loss) or for generating the final classification logits. It should only modify the separation dynamics to prevent off-text dimension expansion.

---

## Component 2: Evidence-Gated Updates (Lever 2)
**Goal:** Filter out overconfident-wrong samples from the prototype moving average update using $s_{max}$ (absolute cosine similarity).

**Implementation Logic:**
1. During the forward pass, compute the absolute cosine similarity between image features $v$ and text embeddings $Z$.
2. For each sample $i$, extract $s_{max, i} = \max_c (\cos(v_i, z_c))$.
3. Define a gating threshold $\tau$ (e.g., the median of $s_{max}$ in the current batch, or a fixed hyperparameter like 0.25 based on empirical distribution).
4. Create a binary mask: $m_i = 1$ if $s_{max, i} \ge \tau$ else $0$.
5. When computing the pseudo-label assignments and updating the moving average of the class prototypes, multiply the sample contributions by $m_i$. Excluded samples should not pull the class prototypes.

---

## Component 3: Stabilized Recirculation & Label-Free Early Stopping (Lever 3)
**Goal:** For the 1K regime, recirculate the batch to increase optimization steps but stop early using $d_{eff\_parallel}$ to prevent drift.

**Implementation Logic:**
1. Wrap the TTA loop for the first 1K samples to repeat for up to 50 epochs (recirculation).
2. At the end of each step, compute $d_{eff\_parallel}$ (the effective rank of the batch features projected via $P_Z$).
   - $d_{eff} = \frac{(\sum \lambda_i)^2}{\sum \lambda_i^2}$, where $\lambda_i$ are the singular values of the covariance matrix of $v_\parallel$.
3. Implement a stopping criterion: Monitor the moving average of $d_{eff\_parallel}$. If the growth plateaus or starts degrading (e.g., $\Delta d_{eff\_parallel} < \epsilon$ for 3 consecutive steps), halt the recirculation and proceed to test/next batch.

## Component 4: Required Validation Script (`validate_projected_batclip.py`)
Write a separate Python diagnostic script to verify the new mechanics. It must include:

**Task D1: Multi-Corruption Sign Flip Check**
- Run the baseline model (50 steps) on `shot_noise` and `impulse_noise`.
- Measure per-step $\rho(d_{eff\_parallel}, \text{acc})$ and $\rho(d_{eff\_global}, \text{acc})$.
- *Expected Output:* Verify if $d_{eff\_parallel}$ remains positively correlated with accuracy while global remains weak/negative across different noise types.

**Task D2: Text Voronoi Volume Check (Sink Class Origin)**
- Sample 100,000 random vectors $u \sim \text{Unif}(S^{d-1})$ on the 512-D unit hypersphere.
- Compute logits $u^\top Z$ and find the argmax class for each vector.
- Calculate the frequency (probability mass) of each class.
- *Expected Output:* If class 3 ("cat") occupies a disproportionately large area of the hypersphere natively, the sink class is a text-prior artifact.

**Task D3: Projected Loss Effectiveness Diagnostic**
- Run a 10K test using the new **Component 1** implementation.
- At each step, measure and log:
  1. The energy of the orthogonal component: $\|v_\perp\|^2 = \|v - v_\parallel\|^2$.
  2. The sink column mass of the pseudo-prototype matrix (from the constrained regression logic in H23).
  3. The rise velocity of $d_{eff\_parallel}$.
- *Expected Output:* $\|v_\perp\|^2$ should not inflate uncontrollably compared to the baseline, and $d_{eff\_parallel}$ should rise faster in the first 5 steps.
