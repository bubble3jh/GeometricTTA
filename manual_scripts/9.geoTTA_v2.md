## 1. 설계 철학 및 절대 원칙 (Design Philosophy)

1. **Feature Space 왜곡 금지:** 피처 공간을 하드 투영($P_Z$)하거나 ZCA 등으로 억지로 구부리는 행위를 전면 금지한다. 기하학적 확장은 텍스트 좌표계인 **로짓(Logit) 공간**에서 수행한다.
2. **Hard Heuristics 금지:** 고정된 임계값($\tau$) 게이팅, 특정 클래스('cat') 타겟팅, Hard OT 할당을 금지한다. 모든 제어는 데이터 기반의 **연속적 가중치(Soft weight)**와 **분포(Distribution)** 기반으로 작동한다.
3. **1-Pass Online 고수:** Recirculation(재순환) 없이, 배치당 1회의 Forward-Backward만으로 모델을 적응시킨다.
4. **수치적 안정성 보장 (Report 15 교훈):** 그래디언트 소멸/폭주, 가중치 조작, 업데이트 진동 등의 치명적 버그를 막기 위해 모든 수식에 명시적인 안전장치(`detach`, `clip`, `MAD scale`)를 적용한다.

## 2. 4대 핵심 Loss 및 수식 (Core Methodology)



### Step A: 분포 정렬 기반 로짓 보정 (Distribution Alignment)
초기 노이즈의 구조적 편향(Sink)을 제거하기 위해, 모델의 과거 예측 분포를 추적하여 로짓에서 사전 확률(Prior)을 깎아냅니다.
* **안전장치:** 자기 상쇄 루프를 막기 위해 히스토그램 $h_t$는 반드시 **'보정 전(Raw) 확률'**로 업데이트하며, 보정량은 클리핑합니다.
  $$q_{raw} = \text{softmax}(\ell^{raw}), \quad h_{t+1} \leftarrow \beta h_t + (1-\beta) \cdot \text{mean}(q_{raw})$$
  $$\Delta_c = \text{clip}(-\log(h_t(c) + \epsilon), [-M, M])$$
  $$\tilde{\ell}_i = \ell_i^{raw} + \lambda_{adj} \Delta \quad \rightarrow \quad \tilde{q}_i = \text{softmax}(\tilde{\ell}_i)$$

### Step B: MAD 기반 Soft Evidence Weighting
OW(Overconfident-Wrong) 샘플의 독성을 걸러내기 위해 절대 신뢰도와 마진을 종합한 부드러운 가중치 $w_i$를 구합니다.
* **안전장치:** Anti-alignment 오작동을 막기 위해 `abs`를 금지하고 최대 코사인 값만 씁니다. 하이퍼파라미터 민감도를 줄이기 위해 MAD(Median Absolute Deviation)로 스케일링하며, 모델의 치팅을 막기 위해 그래디언트를 **`detach`** 합니다.
  $$s_i = \max_c \ell_{ic}^{raw} \quad (\text{abs 제거})$$
  $$\hat{s}_i = \frac{s_i - \text{median}(s)}{\text{MAD}(s) + \epsilon}$$
  $$w_i = \text{detach}\Big( \sigma(\alpha \hat{s}_i) \cdot \sigma(\beta \hat{m}_i) \Big) \quad (\hat{m}_i\text{는 top1-top2 마진의 MAD 스케일링 값})$$

### Step C: Soft 프로토타입 추출 및 Softplus Repulsion
가중치가 부여된 깨끗한 데이터들로 클래스 중심(Prototype)을 구하고 서로 밀어내어 `Var_inter`를 키웁니다.
* **안전장치:** 하드 할당(Argmax) 대신 소프트 할당 $\tilde{q}$를 사용하여 편향을 막고, 그래디언트가 죽어버리는 $\exp(-\gamma)$ 대신 Softplus 마진 손실을 사용합니다.
  $$\bar{v}_k = \text{normalize}\left( \frac{\sum_i w_i \tilde{q}_{ik} v_i}{\sum_i w_i \tilde{q}_{ik} + \epsilon} \right)$$
  $$L_{pot} = \sum_{k \neq l} \text{softplus}\left(\gamma \big(\cos(\bar{v}_k, \bar{v}_l) - m\big)\right)$$



### Step D: Off-diagonal 상관계수 페널티 (Logit Uniformity)
로짓 공간의 공분산 행렬을 정규화하여 텍스트 부분공간 유효 차원(`d_eff_par`)을 안전하게 팽창시킵니다.
* **안전장치:** 스케일 충돌을 막기 위해 로짓을 표준화(Standardize)한 뒤 비대각(Off-diagonal) 성분만 0으로 만듭니다.
  $$\hat{L} = \frac{\tilde{\ell} - \mu_{\tilde{\ell}}}{\sigma_{\tilde{\ell}} + \epsilon}, \quad R = \frac{1}{B}\hat{L}^\top \hat{L}$$
  $$L_{uni} = \sum_{i \neq j} R_{ij}^2$$

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class SoftLogitTTA(nn.Module):
    def __init__(self, model, text_anchors, logit_scale=100.0):
        super().__init__()
        self.model = model
        self.text_anchors = text_anchors
        self.logit_scale = logit_scale
        
        self.register_buffer('running_hist', torch.ones(10) / 10.0)
        self.beta = 0.9
        
        # Hyperparameters
        self.lambda_adj = 1.0
        self.alpha_s = 2.0    
        self.margin_pot = 0.3 
        self.gamma_pot = 10.0 
        self.w_pot = 1.0
        self.w_uni = 0.5
        self.clip_M = 3.0     

    def forward_step(self, images):
        features = F.normalize(self.model.encode_image(images), dim=-1)
        text_anchors = F.normalize(self.text_anchors, dim=-1)
        
        # 1. Raw Logits & Soft Weights
        raw_logits = (features @ text_anchors.T) * self.logit_scale
        
        with torch.no_grad():
            q_raw = F.softmax(raw_logits, dim=-1)
            self.running_hist = self.beta * self.running_hist + (1 - self.beta) * q_raw.mean(dim=0)
            
            # Robust Scaling (s_max & margin)
            s_max, _ = raw_logits.max(dim=-1)
            s_hat = (s_max - s_max.median()) / (torch.median(torch.abs(s_max - s_max.median())) + 1e-6)
            
            top2, _ = torch.topk(raw_logits, 2, dim=-1)
            margin = top2[:, 0] - top2[:, 1]
            m_hat = (margin - margin.median()) / (torch.median(torch.abs(margin - margin.median())) + 1e-6)
            
            w_i = torch.sigmoid(self.alpha_s * s_hat) * torch.sigmoid(self.alpha_s * m_hat)
            # weights are naturally detached due to torch.no_grad()

        # 2. Logit Prior Correction
        adj_penalty = torch.clamp(-torch.log(self.running_hist + 1e-6), -self.clip_M, self.clip_M)
        adj_logits = raw_logits + self.lambda_adj * adj_penalty
        q_adj = F.softmax(adj_logits, dim=-1)

        # 3. BATCLIP Base Losses
        l_ent = -(q_adj * F.log_softmax(adj_logits, dim=-1)).sum(dim=-1).mean()
        l_i2t = batclip_i2t_loss(features, text_anchors, q_adj) # Ensure this matches your utils

        # 4. Soft-Assigned Prototypes & Potential Field
        l_pot = torch.tensor(0.0, device=features.device)
        v_bar, valid_classes = [], []
        for k in range(10):
            mass = (w_i * q_adj[:, k]).sum()
            if mass > 1e-3:
                vk = ((w_i * q_adj[:, k]).unsqueeze(1) * features).sum(dim=0) / mass
                v_bar.append(F.normalize(vk, dim=-1))
                valid_classes.append(k)
        
        if len(valid_classes) > 1:
            v_bar = torch.stack(v_bar)
            cos_matrix = v_bar @ v_bar.T
            mask = ~torch.eye(len(valid_classes), dtype=torch.bool, device=features.device)
            l_pot = F.softplus(self.gamma_pot * (cos_matrix[mask] - self.margin_pot)).mean()

        # 5. Logit Uniformity (Off-diagonal Correlation)
        mu = adj_logits.mean(dim=0)
        sigma = adj_logits.std(dim=0) + 1e-6
        L_hat = (adj_logits - mu) / sigma
        R = (L_hat.T @ L_hat) / features.size(0)
        mask_R = ~torch.eye(10, dtype=torch.bool, device=features.device)
        l_uni = (R[mask_R] ** 2).sum()

        # 6. Total Loss
        loss = l_ent + l_i2t + self.w_pot * l_pot + self.w_uni * l_uni
        
        # Logging for Diagnostic Safety
        # REQUIRED: Log ||g_ent||, ||g_i2t||, ||g_pot||, ||g_uni|| to wandb/json
        
        self.optimizer.zero_grad()
        loss.backward()
        self.optimizer.step()

        return q_adj.argmax(dim=-1)
