## Figure 1: Compression-Collapse Scatter

### 개요

X축 = compression 정도, Y축 = collapse 정도.
15 corruptions이 각각 하나의 점. Clean이 reference 점선.
"Compression이 심할수록 collapse가 심하다"를 한 그림에 보여줌.

### 필요한 데이터

**16 conditions**: 15 corruptions (severity 5) + clean

각 condition에서:
```
1. pairwise_cos: L2-normalized features의 pairwise cosine mean (compression)
2. per_class_count: argmax prediction의 class별 count (K개)  
3. per_class_pbar: softmax prediction의 class별 평균 p̄ (K개)
```

per_class_count와 per_class_pbar에서 나중에 계산할 것들 (post-hoc, 코드에서 안 해도 됨):
```
- cat% = max(per_class_count) / N
- H(p̄) = -Σ p̄_k log p̄_k
- TV(p̄, 1/K) = 0.5 Σ |p̄_k - 1/K|
- Gini(p̄)
- top2_concentration = (top1 + top2) / N
```

### 실행

```python
import torch
import numpy as np
import json
import os

# Setup
model = ...  # Frozen ViT-B/16 CLIP (no adaptation, pretrained weights)
dataset_clean = ...  # CIFAR-10 test set (10000 samples)
corruptions = ['gaussian_noise', 'shot_noise', 'impulse_noise', 
               'defocus_blur', 'glass_blur', 'motion_blur', 'zoom_blur',
               'snow', 'frost', 'fog', 'brightness',
               'contrast', 'elastic_transform', 'pixelate', 'jpeg_compression']

results = {}

for condition in ['clean'] + corruptions:
    if condition == 'clean':
        dataloader = get_clean_loader(batch_size=200)
    else:
        dataloader = get_corrupt_loader(condition, severity=5, batch_size=200)
    
    all_features = []
    all_probs = []
    all_preds = []
    
    model.eval()
    with torch.no_grad():
        for images, labels in dataloader:
            images = images.cuda()
            features = model.encode_image(images)  # (B, D)
            features = features / features.norm(dim=-1, keepdim=True)  # L2 norm
            logits = model.compute_logits(images)  # (B, K)
            probs = torch.softmax(logits, dim=-1)  # (B, K)
            preds = logits.argmax(dim=-1)  # (B,)
            
            all_features.append(features.cpu())
            all_probs.append(probs.cpu())
            all_preds.append(preds.cpu())
    
    all_features = torch.cat(all_features, dim=0)  # (N, D)
    all_probs = torch.cat(all_probs, dim=0)  # (N, K)
    all_preds = torch.cat(all_preds, dim=0)  # (N,)
    
    # 1. Pairwise cosine mean
    # Full pairwise는 O(N²)로 느림. Random subsample 1000개로 근사.
    idx = torch.randperm(len(all_features))[:1000]
    sub = all_features[idx]  # (1000, D), already L2-normed
    cos_matrix = sub @ sub.T  # (1000, 1000)
    # 대각선 제외 평균
    mask = ~torch.eye(1000, dtype=bool)
    pairwise_cos = cos_matrix[mask].mean().item()
    
    # 2. Per-class prediction count
    K = all_probs.shape[1]
    per_class_count = [(all_preds == k).sum().item() for k in range(K)]
    
    # 3. Per-class p̄
    pbar = all_probs.mean(dim=0).tolist()  # (K,)
    
    results[condition] = {
        'pairwise_cos': pairwise_cos,
        'per_class_count': per_class_count,
        'per_class_pbar': pbar,
        'n_samples': len(all_features),
    }

# Save
out_path = os.path.expanduser("~/Lab/v2/notes/figure1_scatter_data.json")
with open(out_path, 'w') as f:
    json.dump(results, f, indent=2)
print(f"Saved to {out_path}")
```

### 주의

- **Model은 frozen (pretrained) 상태.** Adaptation 전의 CLIP.
- 이미 Exp2에서 cos_corrupt를 측정했으면, 거기서 pairwise_cos를 가져올 수 있음. 다만 per_class_count, per_class_pbar는 별도로 필요.
- clean condition은 corruption 없는 CIFAR-10 test set.
- Random subsample 1000개의 pairwise cosine은 full과 거의 동일 (SE < 0.001).

## Figure 2: Adaptation Trajectory

### 개요

X축 = adaptation step, Y축 = 다양한 metrics.
2 methods (TENT, CAMA) × 1 corruption (gaussian_noise) × step별 기록.
Clean/Corrupted 기준선 포함.

### 필요한 데이터

**매 5 step** (총 50 steps → 10 datapoints + step 0):

```
Per step:
  1. per_class_count (K개): argmax class별 count (현재 batch)
  2. per_class_pbar (K개): softmax 평균 (현재 batch)
  3. online_acc: running average accuracy
  4. pairwise_cos: features의 pairwise cosine mean (현재 batch)
  5. I_batch: H(p̄) - mean H(p_i)
  6. loss: 현재 loss 값

매 10 step (cost가 크므로 간격 넓게):
  7. overconf_wrong: max(p_i) > 0.9 AND argmax ≠ y_i인 비율
     → 전체 test set offline eval 필요 (10000 samples)
     → 0, 10, 20, 30, 40, 50 = 6회
```

```python
def compute_pairwise_cos(features_normed, n_subsample=200):
    """features_normed: (B, D), already L2-normalized"""
    B = features_normed.shape[0]
    if B > n_subsample:
        idx = torch.randperm(B)[:n_subsample]
        sub = features_normed[idx]
    else:
        sub = features_normed
    cos = sub @ sub.T
    mask = ~torch.eye(len(sub), dtype=bool, device=cos.device)
    return cos[mask].mean().item()

def compute_overconf_wrong(model, loader, threshold=0.9):
    """전체 test set에서 overconfident-wrong rate 계산"""
    total = 0
    oc_wrong = 0
    model.eval()
    with torch.no_grad():
        for images, labels in loader:
            images, labels = images.cuda(), labels.cuda()
            logits = model.compute_logits(images)
            probs = torch.softmax(logits, dim=-1)
            max_prob, preds = probs.max(dim=-1)
            oc_wrong += ((max_prob > threshold) & (preds != labels)).sum().item()
            total += len(labels)
    return oc_wrong / total
```

## Post-hoc 계산 (plotting 시)

JSON에서 per_class_count와 per_class_pbar를 로드하면 다음을 전부 계산 가능:

```python
import numpy as np

def compute_collapse_metrics(per_class_count, per_class_pbar, K):
    count = np.array(per_class_count)
    pbar = np.array(per_class_pbar)
    N = count.sum()
    
    return {
        'cat_pct': count.max() / N,
        'top2_conc': np.sort(count)[-2:].sum() / N,
        'H_pbar': -(pbar * np.log(pbar + 1e-10)).sum(),
        'TV_uniform': 0.5 * np.abs(pbar - 1/K).sum(),
        'gini': 1 - (pbar ** 2).sum(),
        'max_pbar': pbar.max(),
    }
```

이걸로 Figure 1의 Y축, Figure 2의 Y축 후보를 자유롭게 교체 가능.
