## 실험 설계
- **Corruption**: gaussian_noise only
- **Severity**: 1, 2, 3, 4, 5
- **Dataset**: CIFAR-10-C (K=10, B=200)
- **Backbone**: ViT-B/16 (OpenAI CLIP), frozen (no adaptation for metrics 1-2)
- **총 5 conditions** + clean baseline = 6 points

## 수집할 metrics (각 severity에서)

### Phase 1: Frozen model (adaptation 전)
모델 전혀 건드리지 않고, 전체 test set forward만.

```python
# 1. Compression metric
cos_corrupt = mean pairwise cosine similarity of L2-normalized image features
# 전체 test set (10,000 samples)의 feature를 뽑고,
# random 1000 pairs의 cosine similarity 평균 (또는 전체 평균)

# 2. Collapse metrics (frozen model predictions)
p_bar = (1/N) * sum(softmax(logits))  # 전체 test set marginal
H_pbar = -sum(p_bar * log(p_bar))     # marginal entropy
top1_frac = max(p_bar)                # 가장 많이 예측된 class의 비율
top3_frac = sum(sorted(p_bar)[-3:])   # top-3 class 비율 합
top1_class = argmax(p_bar)            # sink class index
cat_pct = p_bar[3]                    # class 3 (cat) 비율 (참고용)
```

### Phase 2: TENT adaptation
각 severity에서 TENT를 처음부터 돌림.

```python
# 3. Harm metric
tent_online_acc  = online accuracy (전체 test set 순회 후)
tent_offline_acc = offline accuracy (adaptation 끝난 모델로 전체 재평가)

# 참고: ZS accuracy도 Phase 1에서 함께 수집
zs_acc = zero-shot accuracy (frozen model)
```

### Phase 3: Clean baseline (severity 0)
corruption 없는 clean CIFAR-10 test set에서 동일 metrics 수집.

```python
cos_clean = pairwise cosine (clean)
H_pbar_clean = marginal entropy (clean)
zs_acc_clean = zero-shot accuracy (clean)
# TENT on clean은 optional (있으면 좋음)
```

## 주의사항

1. **Clean baseline**: severity=0은 CIFAR-10 original test set 사용
2. **Pairwise cosine**: L2-normalize 후 계산. Random sampling으로 충분 (전체 N²은 과다)
3. **TENT**: 매 severity마다 model을 처음부터 reset 후 돌릴 것
4. **같은 random seed 사용**: 모든 severity에서 동일 seed로 data order 고정
5. **cos_pairwise 계산 시 같은 class끼리만 하지 말 것**: 전체 random pairs (class 무관)
