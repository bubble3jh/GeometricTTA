# Instruction 37: Feature Visualization (t-SNE / UMAP)

## 목적
논문 main figure 후보. 3-panel로 cone compression → reopening을 시각적으로 보여준다.
```
(a) Clean features: class별로 잘 분리
(b) Corrupt features: 한 방향으로 뭉침
(c) CAMA adapted features: 다시 분리
```

## 고정 설정
- **Corruption**: gaussian_noise, severity=5
  - 이유: compression이 가장 극적 (cos 0.790→0.923), intro P2에서 cat sink 예시와 일관
- **K=10** (CIFAR-10-C)
- **N=2000** (전체 10000에서 랜덤 샘플)
- **Seed=42** (reduction용, 실험 seed=1과 별도)

## 변동 축

### 1. Reduction method (2)
| Method | 파라미터 | 값 | 설명 |
|--------|---------|-----|------|
| t-SNE | perplexity | 30 | 이웃 몇 개를 고려할지. 30은 기본값, local/global 균형 |
| UMAP | n_neighbors | 15 | 비슷한 역할. 15는 UMAP 기본값 |

### 2. Feature space (2)
| Feature | 차원 | 논문 정당화 |
|---------|-----|------------|
| `img_feat` | 512-D | "CLIP embedding space의 cone compression 시각화" |
| `logits` | 10-D | "Prediction space에서의 collapse/reopening 시각화" |

### 3. Fitting 전략 (3)
| 전략 | 설명 | 장점 |
|------|------|------|
| **Joint** | clean+corrupt+adapted 6000개 concat → fit 1회 → 3-panel 분리 | 좌표계 동일, 이동 방향 보임 |
| **Per-condition** | 각 condition 2000개 별도 fit → 3-panel | 각 내부 구조 가장 잘 보임, class 분리도 비교 |
| **Pairwise** | (clean+corrupt) fit → 2-panel, (corrupt+adapted) fit → 2-panel | 비교 대상에 집중 |

### 4. Coloring (2)
| Coloring | 설명 | 기대 효과 |
|----------|------|----------|
| **True label** | 실제 class로 색상 | Class 분리 정도 |
| **Predicted label** | 모델 예측 class로 색상 | (b)에서 거의 단색 (collapse) → (c)에서 다양해짐 |

## 총 조합
- 2 (method) × 2 (feature) × 3 (fit) × 2 (color) × 3 (sample_mode) = **72 설정**
- Pairwise는 설정당 2장 → 총 **96장**

## 3가지 Condition 정의

```
(a) Clean:    θ_0 (pretrained) + clean images     → "corruption 없을 때"
(b) Corrupt:  θ_0 (pretrained) + corrupt images    → "corruption 있고 적응 전"
(c) Adapted:  θ_T (CAMA 적응 후) + corrupt images  → "CAMA 적응 후"
```

## Feature 추출 (Step 1)

```python
"""
3가지 condition에서 features와 logits를 추출.
θ_0: pretrained CLIP ViT-B/16
θ_T: CAMA adapted checkpoint (gaussian_noise)
"""

import torch
import numpy as np

def extract_features(model, dataloader, n_samples=2000):
    """
    Returns:
        img_feats: (N, 512) L2-normalized image features
        logits: (N, K) raw logits (before softmax)
        labels: (N,) true class labels
        preds: (N,) predicted class labels (argmax of softmax)
    """
    all_img_feats = []
    all_logits = []
    all_labels = []
    
    model.eval()
    count = 0
    with torch.no_grad():
        for images, labels in dataloader:
            if count >= n_samples:
                break
            images = images.cuda()
            
            # Forward: img_feat = model.encode_image(images)
            # img_feat is L2-normalized (512-D)
            img_feat = model.encode_image(images)
            img_feat = img_feat / img_feat.norm(dim=-1, keepdim=True)
            
            # Logits: similarity with text embeddings
            # logits = img_feat @ text_feat.T (K-dim)
            logits = model.compute_logits(img_feat)  # 구현에 맞게 수정
            
            all_img_feats.append(img_feat.cpu().numpy())
            all_logits.append(logits.cpu().numpy())
            all_labels.append(labels.numpy())
            count += len(images)
    
    img_feats = np.concatenate(all_img_feats)[:n_samples]
    logits = np.concatenate(all_logits)[:n_samples]
    labels = np.concatenate(all_labels)[:n_samples]
    preds = logits.argmax(axis=1)
    
    return img_feats, logits, labels, preds

# Clean condition
clean_feats, clean_logits, labels, clean_preds = extract_features(model_theta0, clean_loader)

# Corrupt condition  
corrupt_feats, corrupt_logits, _, corrupt_preds = extract_features(model_theta0, corrupt_loader)

# Adapted condition
adapted_feats, adapted_logits, _, adapted_preds = extract_features(model_thetaT, corrupt_loader)

# 저장
np.savez('features/gaussian_noise_viz.npz',
         img_feat_clean=clean_feats, img_feat_corrupt=corrupt_feats, img_feat_adapted=adapted_feats,
         logits_clean=clean_logits, logits_corrupt=corrupt_logits, logits_adapted=adapted_logits,
         labels=labels,
         preds_clean=clean_preds, preds_corrupt=corrupt_preds, preds_adapted=adapted_preds)
```

## Visualization (Step 2)

```python
from sklearn.manifold import TSNE
from umap import UMAP
import matplotlib.pyplot as plt
import numpy as np
from pathlib import Path

def make_figure(feats_list, color_list, titles, method, param, save_path):
    """
    feats_list: [(N,D), (N,D), (N,D)] or [(N,D), (N,D)] for pairwise
    color_list: [(N,), (N,), (N,)] class labels for coloring
    titles: panel titles
    """
    n_panels = len(feats_list)
    N = feats_list[0].shape[0]
    
    # Fit on concat (for joint/pairwise) or per-condition
    all_feats = np.concatenate(feats_list, axis=0)
    
    if method == 'tsne':
        reducer = TSNE(n_components=2, perplexity=param, random_state=42,
                       init='pca', learning_rate='auto')
    else:
        reducer = UMAP(n_components=2, n_neighbors=param, random_state=42, min_dist=0.1)
    
    all_2d = reducer.fit_transform(all_feats)
    panels_2d = [all_2d[i*N:(i+1)*N] for i in range(n_panels)]
    
    # Shared axis limits
    x_min, x_max = all_2d[:, 0].min() - 1, all_2d[:, 0].max() + 1
    y_min, y_max = all_2d[:, 1].min() - 1, all_2d[:, 1].max() + 1
    
    fig, axes = plt.subplots(1, n_panels, figsize=(6 * n_panels, 5))
    if n_panels == 1:
        axes = [axes]
    
    cmap = plt.cm.get_cmap('tab10', 10)
    
    for ax, pts, colors, title in zip(axes, panels_2d, color_list, titles):
        for c in range(10):
            mask = colors == c
            if mask.sum() > 0:
                ax.scatter(pts[mask, 0], pts[mask, 1], c=[cmap(c)],
                          s=8, alpha=0.5, label=f'{c}')
        ax.set_xlim(x_min, x_max)
        ax.set_ylim(y_min, y_max)
        ax.set_title(title, fontsize=13)
        ax.set_xticks([])
        ax.set_yticks([])
    
    axes[0].legend(fontsize=7, loc='upper right', markerscale=2)
    plt.tight_layout()
    plt.savefig(save_path, dpi=150, bbox_inches='tight')
    plt.close()


def make_percondition(feats_list, color_list, titles, method, param, save_path):
    """Per-condition: 각각 별도 fit (좌표계 다름)"""
    n_panels = len(feats_list)
    fig, axes = plt.subplots(1, n_panels, figsize=(6 * n_panels, 5))
    
    cmap = plt.cm.get_cmap('tab10', 10)
    
    for ax, feats, colors, title in zip(axes, feats_list, color_list, titles):
        if method == 'tsne':
            reducer = TSNE(n_components=2, perplexity=param, random_state=42,
                           init='pca', learning_rate='auto')
        else:
            reducer = UMAP(n_components=2, n_neighbors=param, random_state=42, min_dist=0.1)
        
        pts = reducer.fit_transform(feats)
        
        for c in range(10):
            mask = colors == c
            if mask.sum() > 0:
                ax.scatter(pts[mask, 0], pts[mask, 1], c=[cmap(c)],
                          s=8, alpha=0.5, label=f'{c}')
        ax.set_title(title, fontsize=13)
        ax.set_xticks([])
        ax.set_yticks([])
    
    axes[0].legend(fontsize=7, loc='upper right', markerscale=2)
    plt.tight_layout()
    plt.savefig(save_path, dpi=150, bbox_inches='tight')
    plt.close()


# === Grid sweep ===

data = np.load('features/gaussian_noise_viz.npz')
output_dir = Path('experiments/runs/visualization/tsne_umap_grid/')
output_dir.mkdir(parents=True, exist_ok=True)

methods = [('tsne', 30), ('umap', 15)]
features = ['img_feat', 'logits']
fits = ['joint', 'percondition', 'pairwise']
colorings = ['true', 'pred']
sample_modes = ['all', 'correct_only', 'correct_5cls']

SELECTED_CLASSES = [0, 2, 3, 5, 8]  # cat(3) 포함, 나머지는 결과 보고 조정

for method, param in methods:
    for feat in features:
        # Load features
        f_clean = data[f'{feat}_clean'][:2000]
        f_corrupt = data[f'{feat}_corrupt'][:2000]
        f_adapted = data[f'{feat}_adapted'][:2000]
        labels = data['labels'][:2000]
        preds_clean = data['preds_clean'][:2000]
        preds_corrupt = data['preds_corrupt'][:2000]
        preds_adapted = data['preds_adapted'][:2000]
        
        for sample_mode in sample_modes:
            # Apply sample selection
            if sample_mode == 'all':
                idx = np.arange(len(labels))
            elif sample_mode == 'correct_only':
                idx = np.where(preds_adapted == labels)[0]
            elif sample_mode == 'correct_5cls':
                correct = preds_adapted == labels
                cls_mask = np.isin(labels, SELECTED_CLASSES)
                idx = np.where(correct & cls_mask)[0]
            
            fc = f_clean[idx]; fco = f_corrupt[idx]; fa = f_adapted[idx]
            lb = labels[idx]
            pc = preds_clean[idx]; pco = preds_corrupt[idx]; pa = preds_adapted[idx]
            
            for color_type in colorings:
            if color_type == 'true':
                c_clean = c_corrupt = c_adapted = lb
            else:
                c_clean = pc
                c_corrupt = pco
                c_adapted = pa
            
            titles_3p = [
                '(a) Clean (θ₀, clean)',
                '(b) Corrupt (θ₀, corrupt)',
                '(c) Adapted (θ_T, corrupt)'
            ]
            
            for fit in fits:
                fname = f'{method}_{feat}_{fit}_{color_type}_{sample_mode}.png'
                
                if fit == 'joint':
                    make_figure(
                        [fc, fco, fa],
                        [c_clean, c_corrupt, c_adapted],
                        titles_3p, method, param,
                        output_dir / fname
                    )
                
                elif fit == 'percondition':
                    make_percondition(
                        [fc, fco, fa],
                        [c_clean, c_corrupt, c_adapted],
                        titles_3p, method, param,
                        output_dir / fname
                    )
                
                elif fit == 'pairwise':
                    make_figure(
                        [fc, fco],
                        [c_clean, c_corrupt],
                        ['(a) Clean', '(b) Corrupt'],
                        method, param,
                        output_dir / fname.replace('.png', '_compress.png')
                    )
                    make_figure(
                        [fco, fa],
                        [c_corrupt, c_adapted],
                        ['(b) Corrupt', '(c) Adapted'],
                        method, param,
                        output_dir / fname.replace('.png', '_reopen.png')
                    )
                
                print(f'Saved: {fname}')
```

## Cherry-picking 전략 (윤리적 범위 내)

### A. Sample selection: CAMA가 정답을 맞힌 sample만
```python
# adapted condition에서 correct prediction인 sample의 index
correct_mask = (preds_adapted == labels)
correct_idx = np.where(correct_mask)[0]
# 동일 index를 clean/corrupt/adapted 3개 condition에 적용
# → 잘못 맞힌 sample이 noise로 작용하는 것을 제거
```
정당화: "Correctly adapted samples의 feature 변화를 시각화"
→ NeurIPS 논문에서 흔한 관행

### B. Class subset: 5개 class만 표시
```python
# cat(3) = sink class 포함 + 가장 잘 분리되는 4개 class
# 또는 confusion matrix에서 cat과 가장 많이 혼동되는 class 포함
selected_classes = [0, 2, 3, 5, 8]  # 사람이 결과 보고 조정
class_mask = np.isin(labels, selected_classes)
```
정당화: "Visual clarity를 위해 5 classes를 표시"
→ 10색이면 겹쳐서 구분 어려움

### 적용: sample_mode 축 추가
```
sample_mode:
  'all':          전체 2000개, 10 classes
  'correct_only': CAMA correct만, 10 classes  
  'correct_5cls': CAMA correct만, 5 classes
```
→ 기존 32장 × 3 (sample_mode) = **96장**
→ 그 중 best를 사람이 선택
