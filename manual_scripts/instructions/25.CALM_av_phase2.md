## Core Idea

Instead of:
```
L = mean(l_ent_i) + λ·KL(p̄‖π)
```

Use:
```
L = mean(a_i · l_ent_i) + λ·KL(p̄‖π)
```

where **a_i is detached** (no gradient through gate).
Trusted samples (a_i > 1) get more entropy minimization; untrusted (a_i < 1) get less.

Since mean(a_i) ≈ 1, the weighted L_ent has similar magnitude to unweighted.

## Sample Gate Formula

```python
t̃_i   = normalize(w_i @ T)            # (B, D) harmonic text mixture
cos_f_tmix = (f_i · t̃_i)              # (B,) alignment to personal mixture
mean_cos   = mean_k(f_i · t_k)         # (B,) mean alignment to all classes
u_i        = max(0, cos_f_tmix − mean_cos)  # (B,) excess alignment
a_i        = u_i / mean(u + ε)         # (B,) normalised, mean ≈ 1
```
