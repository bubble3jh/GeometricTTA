## 실험 1: b̂_k vs log π_k Rank Correlation (15-corruption)

**목적:** Evidence prior π가 corruption의 logit bias를 실제로 추정하는지 검증.

**방법:** 각 corruption에서 첫 batch(BS=200) forward 1회.

```python
for corruption in 15_corruptions:
    logits = model(first_batch)  # shape: (B, K), 초기 모델 θ_0
    
    # Corruption logit bias (batch 평균)
    b_hat = logits.mean(dim=0)  # shape: (K,)
    
    # Evidence prior
    pi = compute_evidence_prior(logits)  # 기존 함수 사용
    log_pi = torch.log(pi)
    
    # Sink class 확인
    sink_from_b = b_hat.argmax().item()
    sink_from_pi = pi.argmax().item()
    sink_from_pbar = softmax(logits).mean(dim=0).argmax().item()
    
    # Rank correlation
    from scipy.stats import spearmanr, pearsonr
    spearman_r, spearman_p = spearmanr(b_hat.cpu().numpy(), log_pi.cpu().numpy())
    pearson_r, pearson_p = pearsonr(b_hat.cpu().numpy(), log_pi.cpu().numpy())
    
    log(corruption, b_hat, log_pi, sink_from_b, sink_from_pi, sink_from_pbar,
        spearman_r, pearson_r)
```

**출력 (per corruption):**
```json
{
  "corruption": "gaussian_noise",
  "b_hat": [K values],
  "log_pi": [K values],
  "sink_class_from_b": int,
  "sink_class_from_pi": int,
  "sink_class_from_pbar": int,
  "sink_match": true/false,
  "spearman_r": float,
  "pearson_r": float
}
```

**Summary table:** 15-corruption별 spearman_r + sink match 여부.

**판정:** spearman_r > 0.8 (다수 corruption) + sink match 일관 → π가 bias 추정.

**예상 시간:** ~5분.

## 실험 2: Cone Compression — CALM 기준 재측정 (15-corruption)

**목적:** CALM adaptation 전후의 pairwise cosine similarity 측정. 기존 데이터는 H2 기준이라 CALM으로 재실험 필요.

**방법:** 각 corruption에서 clean features, corrupt features, CALM-adapted features의 pairwise cos 측정.

```python
for corruption in 15_corruptions:
    # Clean features (corruption 없는 원본)
    clean_feats = model.image_encoder(clean_images)  # θ_0 기준
    cos_clean = pairwise_cosine(clean_feats).mean().item()
    
    # Corrupt features (adaptation 전, θ_0)
    corrupt_feats = model.image_encoder(corrupt_images)  # θ_0 기준
    cos_corrupt = pairwise_cosine(corrupt_feats).mean().item()
    
    # CALM-adapted features (adaptation 후, θ_T)
    # → 50 step adaptation 실행 후 측정
    adapted_feats = adapted_model.image_encoder(corrupt_images)  # θ_T
    cos_adapted = pairwise_cosine(adapted_feats).mean().item()
    
    log(corruption, cos_clean, cos_corrupt, cos_adapted)
```

pairwise_cosine: batch 내 모든 feature 쌍의 cosine similarity 평균.

```python
def pairwise_cosine(feats):
    # feats: (B, D), normalized
    feats = F.normalize(feats, dim=1)
    sim = feats @ feats.T  # (B, B)
    mask = ~torch.eye(B, dtype=bool, device=sim.device)
    return sim[mask].mean()
```

**출력:**
```json
{
  "corruption": "gaussian_noise",
  "cos_clean": float,
  "cos_corrupt": float,  
  "cos_calm_adapted": float,
  "cone_opened": cos_corrupt - cos_calm_adapted
}
```

**Summary table:** 15-corruption별 3개 cos 값.

**주의:** Clean images는 CIFAR-10 test set (corruption 없는 원본). 같은 이미지의 clean/corrupt 버전 비교.

**예상 시간:** ~20분 (15 corruption × adaptation 50 step).

## 실험 3: Proposition A.1 u 측정

**목적:** Soft-to-hard accuracy gap u = avg(1 - max_k p_ik) 정량화.

**방법:** CALM adaptation 후 prediction에서 측정. 별도 adaptation 불필요 — 실험 2의 adapted model 재활용.

```python
for corruption in 15_corruptions:
    # Adapted model의 prediction
    logits = adapted_model(corrupt_images)
    probs = softmax(logits)
    
    max_probs = probs.max(dim=1).values  # shape: (B,)
    u = (1 - max_probs).mean().item()
    
    # 추가 지표
    mean_entropy = -(probs * torch.log(probs + 1e-10)).sum(dim=1).mean().item()
    
    log(corruption, u, mean_entropy)
```

**출력:**
```json
{
  "corruption": "gaussian_noise",
  "u_soft_hard_gap": float,
  "mean_entropy_adapted": float
}
```

**판정:** u < 0.05이면 "adapted prediction이 충분히 sharp하여 soft ceiling ≈ hard ceiling".

**예상 시간:** ~5분 (실험 2와 병합 가능, 추가 forward만).

## 실험 4: KL(p̄_t ‖ p†) Trajectory 추적

**목적:** Adaptation 중 batch marginal p̄가 reference equilibrium p†에 가까워지는지 확인. "Equilibrium correction이 실제로 일어나는가" 시각화.

**방법:** 단일 corruption (gaussian_noise)에서 50 step adaptation 중 매 step 로깅.

```python
# p† 계산 (step 0에서 1회)
alpha = lambda_auto / (lambda_auto - 1)
log_p_dag = alpha * torch.log(pi)
log_p_dag -= log_p_dag.max()
p_dag = torch.exp(log_p_dag)
p_dag = p_dag / p_dag.sum()

trajectory = []
for step in range(50):
    logits = model(batch)
    probs = softmax(logits)
    p_bar = probs.mean(dim=0)
    
    # KL(p̄ ‖ p†)
    kl_to_pdag = F.kl_div(torch.log(p_bar + 1e-10), p_dag, reduction='sum').item()
    
    # KL(p̄ ‖ uniform) for comparison
    kl_to_uniform = F.kl_div(torch.log(p_bar + 1e-10), 
                              torch.ones(K)/K, reduction='sum').item()
    
    # TV(p̄, θ) — θ = 1/K for balanced
    tv_to_theta = 0.5 * (p_bar - 1.0/K).abs().sum().item()
    
    trajectory.append({
        "step": step,
        "kl_to_pdag": kl_to_pdag,
        "kl_to_uniform": kl_to_uniform,
        "tv_to_theta": tv_to_theta,
        "p_bar": p_bar.cpu().tolist(),
        "online_acc": acc
    })
    
    # Normal adaptation step
    loss = ...
    loss.backward()
    optimizer.step()
```

**출력:**
```json
{
  "corruption": "gaussian_noise",
  "p_dag": [K values],
  "trajectory": [
    {"step": 0, "kl_to_pdag": float, "kl_to_uniform": float, "tv_to_theta": float, "online_acc": float},
    ...
  ]
}
```

**기대:** KL(p̄ ‖ p†)가 step 증가에 따라 감소 → p̄가 p†에 수렴.

**예상 시간:** ~10분.
